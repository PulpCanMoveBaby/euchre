# Classic Euchre - Right in Your Terminal

  Test your Euchre skills against computer players using algorithmic strategy. 
  The advanced trick-taking logic of the players make this a fun and challenging way to hone and/or improve your Euchre skills.
  Or if you're just looking to kick back and play, you will find all the Euchre your heart desires right here.
  You have come to the right place!

  But beware beginner to intermediate players, these rule-based AI players will take no mercy on you!

## Features

  - Play against AI computer players using rule-based logic
  - Real-time scoring and tricks displayed so you can easily plan your attack
  - Simple guided instructions throughout gameplay
  - Simulation mode:
      - choose a number of games to be played by four computer players 
      - results are logged and displayed

## Getting Started

  - Python 3.x installed on your system
  - That's literally it

## Installation Steps
  - Clone the repo:
      - git clone https://github.com/PulpCanMoveBaby/euchre.git
  - Run the game:
      - In your terminal, navigate to the directory where the repo was cloned and run (one of the two should work):
          - python3 euchre.py
          - python euchre.py

## Usage
  - Once lauched, follow the instructions on the screen and enjoy some Euchre
